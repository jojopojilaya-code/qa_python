ОПИСАНИЕ

--------

Тестирование приложения BooksCollector.



Программа позволяет:

\- добавлять книги;

\- назначать жанры;

\- получать список книг по жанру;

\- отбирать книги для детей;

\- добавлять и удалять книги из избранного.



Все основные функции приложения покрыты автотестами.





СОДЕРЖИМОЕ КАТАЛОГА

------------------

main.py

&nbsp;   Основной файл программы. Содержит класс BooksCollector.



test\_books\_collector.py

&nbsp;   Файл с автотестами для проверки работы программы.





СПИСОК ТЕСТОВ

-------------

1\. Проверка, что книга с пустым названием не добавляется.

2\. Проверка, что книга с названием длиннее 40 символов не добавляется.

3\. Проверка, что у новой книги жанр пустой.

4\. Проверка установки жанра существующей книге.

5\. Проверка, что некорректный жанр не устанавливается.

6\. Проверка получения книг по заданному жанру.

7\. Проверка получения списка книг для детей.

8\. Проверка добавления книги в избранное и удаления из него.

9\. Проверка удаления книги, отсутствующей в избранном.

10\. Проверка, что нельзя добавить в избранное несуществующую книгу.
